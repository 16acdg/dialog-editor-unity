Custom Dialog Trees Made Easy


Part of Unity's scripting API involves extending their EditorWindow class to create your own custom GUI windows that work as if they are part of the default Game Engine editor. A task I recently undertook for a videogame I'm building involved creating a dialog system. Most of the dialog system was very straightforward to implement, however creating a dialog tree for a character involved hardcoding an array of DialogNode objects, holding dialog prompts and user responses. Due to the large number of characters in my game, and to enable additional use for future projects, I elected to make a more scalable and user-friendly dialog tree creation tool. This involved developing a GUI that would display the dialog tree within the Unity editor. You can see a sample of the editor window below.

I elected to make this from scratch to practice using the Unity EditorWindow class to make my own editor windows, and to produce a simple, free-to-use alternative to similar plugins already on the Unity Asset Store. Sometime soon in the future I hope to release this plugin to the public through the Unity Asset Store.
